## 🛠️ Common Troubleshooting

Most users find the installation simple, but errors happen. Review these tips if you run into problems during setup or usage.

*   **Installer blocked:** Windows SmartScreen might block the file because it does not recognize the publisher. If this happens, click "More info," then click "Run anyway."
*   **App won't open:** Right-click the shortcut icon on your desktop and select "Run as administrator."
*   **Video playback lag:** Large files require many resources. If your playback skips, try lowering the preview quality in the playback settings menu. This does not change the quality of your final exported video.
*   **Drivers missing:** Video editors rely on your graphics card. Visit the website for your graphics card manufacturer—either NVIDIA, AMD, or Intel—and download the latest drivers for your specific hardware.
*   **Storage errors:** Ensure your hard drive has enough space. A common mistake involves trying to export a long video to a drive that is nearly full.

## 📁 File Management

Proper file management keeps your projects safe. Keep your original footage, audio tracks, and project files in a single folder relative to each other. If you move a file after you import it into the software, the program loses the connection. You would then need to re-link every file.

Always save your project often. Use the keyboard shortcut Ctrl + S to save your changes after every edit. This protects you against software crashes or power loss.
